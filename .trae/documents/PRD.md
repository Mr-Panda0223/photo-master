# 安卓图片编辑APP - 产品需求文档 (PRD)

## 1. 产品概述

### 1.1 产品定位与价值主张

**产品名称**：PhotoMaster（暂定）

**产品定位**：一款面向Android平台的纯本地图片编辑工具，专注于为普通用户提供简洁易用的图片美化功能，所有数据处理均在本地完成，无需联网，保护用户隐私。

**核心价值主张**：
- **完全本地处理**：所有图片处理均在本地完成，无需网络连接，不上传任何数据
- **零用户信息**：不存储任何用户信息，无需注册登录，安装即用
- **简洁易用**：面向普通用户设计，操作简单直观，无需专业知识
- **隐私保护**：用户图片永远不会离开设备，最大程度保护隐私

### 1.2 目标用户画像

| 用户类型 | 特征描述 | 核心需求 | 使用场景 |
|---------|---------|---------|---------|
| **普通用户** | 日常拍照、社交分享需求 | 快速美化、一键滤镜、简单裁剪 | 朋友圈配图、生活记录 |
| **轻度编辑需求者** | 偶尔需要修图但不想学习复杂软件 | 简单调整、添加文字、基础滤镜 | 旅行照片、美食分享 |
| **隐私敏感用户** | 注重个人隐私，不信任云端处理 | 本地处理、无账号系统、数据不上传 | 个人照片处理 |

### 1.3 核心使用场景

1. **日常修图**：用户拍摄照片后，快速调整亮度、对比度，添加滤镜后保存
2. **社交分享准备**：对照片进行简单美化、裁剪为合适比例、添加文字说明
3. **照片整理**：批量调整照片色调，统一风格后保存到本地相册
4. **简单创意**：为照片添加贴纸、文字、简单涂鸦

## 2. 商业模式

### 2.1 分发模式

**限时使用机制**：
- 每个安装包有效期为7天
- 用户可免费使用全部功能7天
- 7天后应用提示用户下载新的安装包
- 开发者每周定期更新安装包下载链接，旧链接作废

**优势**：
- 无需服务端存储用户信息
- 无需账号系统和付费验证
- 简化的商业模式，降低开发和维护成本
- 定期更新保证用户获取最新版本

### 2.2 用户获取流程

1. 用户从开发者提供的链接下载安装包
2. 安装后可直接使用全部功能
3. 应用启动时检查有效期，显示剩余天数
4. 到期前3天开始提示用户准备更新
5. 到期后引导用户获取新的下载链接

## 3. 核心功能

### 3.1 功能模块

本应用包含以下核心页面：

1. **首页/相册页**：本地相册浏览、图片选择
2. **编辑页**：核心图片编辑工作区，包含所有编辑工具
3. **导出页**：导出设置、格式选择、保存路径
4. **设置页**：应用偏好设置、存储管理

### 3.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| **首页** | 相册浏览 | 读取本地相册，按时间/文件夹分类展示缩略图 |
| **首页** | 快速操作 | 拍照入口、从其他应用接收图片 |
| **编辑页** | 工具栏 | 底部工具栏：裁剪、调整、滤镜、光影、文字、画笔 |
| **编辑页** | 画布区域 | 主编辑区域，支持双指缩放、平移、撤销/重做手势 |
| **编辑页** | 参数面板 | 滑动调节各项参数，实时预览效果 |
| **导出页** | 格式选择 | 支持JPEG/PNG格式，质量调节 |
| **导出页** | 尺寸设置 | 原始尺寸、自定义尺寸、预设比例 |
| **导出页** | 保存选项 | 保存到相册、指定文件夹 |
| **设置页** | 偏好设置 | 默认导出格式、自动保存间隔、界面主题 |
| **设置页** | 存储管理 | 缓存清理、草稿管理、存储空间统计 |
| **设置页** | 有效期信息 | 显示当前版本剩余有效期 |

## 4. 核心流程

### 4.1 用户操作流程

**标准用户流程**：
1. 打开APP → 浏览相册 → 选择图片 → 进入编辑页
2. 选择编辑工具（如滤镜）→ 调整参数 → 实时预览效果
3. 完成编辑 → 进入导出页 → 选择格式/质量 → 保存到本地

### 4.2 页面导航流程

```
启动页 → 首页/相册页 → 编辑页 → 导出页 → 保存到本地
              ↓
           设置页
```

## 6. 非功能需求

### 6.1 性能要求

| 指标 | 要求 | 说明 |
|------|------|------|
| 启动时间 | ≤ 2秒 | 从点击图标到首页完全加载 |
| 图片加载 | ≤ 1秒 | 1080P图片从选择到显示在编辑区 |
| 实时预览 | ≥ 30fps | 调整参数时实时预览帧率 |
| 导出速度 | ≤ 5秒 | 1080P图片导出（质量90%） |
| 内存占用 | ≤ 300MB | 编辑单张1080P图片时的峰值内存 |
| 大图片支持 | ≤ 4K | 支持编辑4K分辨率图片 |

### 6.2 兼容性要求

| 项目 | 要求 |
|------|------|
| Android版本 | 最低Android 8.0（API 26），推荐Android 10+ |
| 设备适配 | 支持手机和平板，适配不同屏幕尺寸和比例 |
| 处理器 | 支持ARM64和ARMv7架构 |
| 存储空间 | 安装包≤50MB，运行时缓存≤200MB |
| 权限需求 | 存储权限（读取/保存图片）、相机权限（拍照功能） |

### 6.3 用户体验要求

| 项目 | 要求 |
|------|------|
| 操作响应 | 所有操作响应时间≤100ms，复杂操作需显示加载状态 |
| 撤销/重做 | 支持至少30步历史记录，支持手势快速撤销 |
| 自动保存 | 每30秒自动保存编辑进度，防止意外退出丢失 |
| 手势支持 | 双指缩放/平移、双击还原缩放 |
| 深色模式 | 支持系统深色模式适配 |
| 多语言 | 支持简体中文 |

## 7. 竞品分析

### 7.1 竞品对比表

| 功能/产品 | **Snapseed** | **美图秀秀** | **PhotoMaster（本APP）** |
|-----------|-------------|-------------|------------------------|
| **定位** | 专业级修图 | 人像美化+社交 | 简洁本地编辑 |
| **本地处理** | ✅ 完全本地 | ❌ 需联网 | ✅ 完全本地 |
| **联网需求** | ❌ 不需要 | ✅ 需要 | ❌ 不需要 |
| **账号系统** | ❌ 不需要 | ✅ 需要 | ❌ 不需要 |
| **用户隐私** | ✅ 高 | ❌ 低 | ✅ 极高 |
| **功能复杂度** | 中等 | 简单 | 简单 |
| **滤镜数量** | 30+ | 50+ | 20+ |

### 7.2 竞品优劣势分析

**Snapseed（Google）**
- **优势**：完全免费无广告、专业级调整工具、操作流畅
- **劣势**：已停止更新、无社交分享、界面略显陈旧
- **差异化机会**：持续更新维护、更简洁的操作流程

**美图秀秀**
- **优势**：人像美化效果自然、模板贴纸丰富
- **劣势**：过度美颜、需要联网、隐私担忧、广告较多
- **差异化机会**：完全本地处理、隐私保护、简洁无广告

### 7.3 我们的竞争优势

1. **完全离线**：无需任何网络连接，飞机上、地铁里都能使用
2. **零隐私风险**：不收集任何用户数据，图片永远不会离开设备
3. **简洁高效**：去除社交、云同步等复杂功能，专注图片编辑本身
4. **无广告干扰**：纯净的使用体验，没有广告打扰

## 8. 技术约束

### 8.1 技术架构原则

| 约束项 | 要求 |
|--------|------|
| 完全离线 | 应用运行时不需要任何网络连接 |
| 本地处理优先 | 所有图片处理算法在本地执行 |
| 原生开发 | Android端使用Kotlin原生开发，确保性能 |
| 图像引擎 | 基于OpenGL ES的GPU加速图像处理 |
| 存储策略 | 使用Android Scoped Storage，适配Android 10+存储权限变化 |
| 内存管理 | 大图分块处理，避免OOM；使用图片缓存池复用内存 |

### 8.2 图片处理能力

| 项目 | 要求 |
|------|------|
| 最大分辨率 | 最大4096×4096或设备支持上限 |
| 色彩深度 | 支持8bit色彩深度处理 |
| 色彩空间 | 支持sRGB |
| 处理精度 | 调整参数支持1级精度 |

### 8.3 有效期验证机制

| 项目 | 要求 |
|------|------|
| 验证方式 | 应用内置有效期，每次启动检查 |
| 提示策略 | 到期前3天开始提示，每天显示剩余天数 |
| 到期处理 | 到期后显示更新提示，提供获取新版本的指引 |
| 防破解 | 采用代码混淆和简单加密保护有效期验证逻辑 |

## 9. 用户界面设计

### 9.1 设计风格

| 设计元素 | 规范 |
|---------|------|
| **整体风格** | 现代简约、清新明快、浅色为主 |
| **主色调** | 白色背景(#FFFFFF) + 强调色青色(#00D4AA) |
| **辅助色** | 浅灰(#F5F5F5)、中灰(#999999)、深灰(#333333) |
| **按钮样式** | 圆角矩形（8dp圆角），主按钮填充色，次按钮描边 |
| **字体** | 中文：系统默认字体；字号层级：标题18sp、正文14sp、辅助12sp |
| **图标风格** | 线性图标（2dp描边），统一24dp尺寸 |
| **布局风格** | 底部导航+底部工具栏，最大化画布区域 |

### 9.2 页面设计概述

| 页面名称 | 模块名称 | UI元素描述 |
|---------|---------|-----------|
| **首页** | 顶部栏 | Logo + 设置按钮，高度56dp，白色背景 |
| **首页** | 相册网格 | 3列网格布局，间距4dp，圆角缩略图 |
| **首页** | 有效期提示 | 顶部或底部显示剩余有效期 |
| **编辑页** | 顶部栏 | 返回/撤销/重做/导出按钮，显示当前图片名称 |
| **编辑页** | 画布区域 | 全屏居中，双指缩放支持，灰色背景衬托 |
| **编辑页** | 底部工具栏 | 横向滚动工具列表，图标+文字标签，选中高亮 |
| **编辑页** | 参数面板 | 底部滑出面板，滑块+数值输入，实时预览 |
| **导出页** | 预览区 | 大图预览，显示导出尺寸和预估文件大小 |
| **导出页** | 设置区 | 格式选择（按钮组）、质量滑块、尺寸选项 |
| **导出页** | 操作区 | 保存按钮，底部固定 |
| **设置页** | 有效期信息 |  prominently显示当前版本有效期和更新指引 |

### 9.3 响应式设计

- **手机端**：竖屏为主，底部工具栏，侧滑面板
- **平板端**：横屏适配，左右分栏布局，更大画布区域

### 9.4 交互细节

| 交互 | 设计 |
|------|------|
| 手势操作 | 双指缩放/平移画布、双击还原、左滑撤销 |
| 参数调节 | 横向滑动调节，数值实时显示，支持点击输入精确值 |
| 工具切换 | 底部工具栏点击切换，平滑过渡动画（200ms） |
| 预览对比 | 长按画布显示原图，松手显示效果，便于对比 |
| 加载状态 | 骨架屏+进度条，避免白屏等待 |
