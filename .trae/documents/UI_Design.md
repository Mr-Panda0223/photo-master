# PhotoMaster - UI/UX设计文档

## 1. 设计概述

### 1.1 设计理念

**简洁至上**：界面元素精简，让用户专注于图片编辑本身  
**直观易用**：操作逻辑符合用户直觉，无需学习成本  
**现代清新**：采用浅色主题，营造轻松愉悦的编辑氛围  
**反馈及时**：每个操作都有清晰的视觉反馈

### 1.2 设计原则

1. **内容优先**：最大化图片编辑区域，工具元素不遮挡主要内容
2. **层级清晰**：通过颜色、大小、间距建立清晰的信息层级
3. **一致性**：相同功能保持一致的交互方式和视觉表现
4. **容错性**：提供撤销/重做功能，允许用户随时回退操作

## 3. 页面设计

### 3.1 首页/相册页

#### 页面结构
```
┌─────────────────────────────────┐
│  PhotoMaster        [设置]      │  ← 顶部栏 (56dp)
├─────────────────────────────────┤
│                                 │
│    ┌────┐ ┌────┐ ┌────┐        │
│    │    │ │    │ │    │        │  ← 相册网格
│    └────┘ └────┘ └────┘        │     (3列，间距4dp)
│    ┌────┐ ┌────┐ ┌────┐        │
│    │    │ │    │ │    │        │
│    └────┘ └────┘ └────┘        │
│                                 │
├─────────────────────────────────┤
│  [相机]  [相册]  [导入]         │  ← 底部快捷操作 (56dp)
└─────────────────────────────────┘
```

#### 元素说明

**顶部栏**
- 高度：56dp
- 背景：白色 + 底部阴影(1dp)
- 左侧：Logo文字 "PhotoMaster" (18sp, Medium, 主色)
- 右侧：设置图标按钮 (24dp)

**有效期提示条**（顶部栏下方）
- 高度：40dp
- 背景：警告黄(#FAAD14) 10%透明度
- 内容："剩余有效期：X天" (14sp, 警告黄)
- 右侧：更新按钮 (可选)

**相册网格**
- 布局：3列网格
- 间距：4dp
- 缩略图：圆角8dp，比例1:1
- 选中状态：主色边框(2dp) + 主色浅背景
- 日期分组：显示拍摄日期 (12sp, 文字辅助)

**底部快捷操作栏**
- 高度：56dp
- 背景：白色 + 顶部阴影(1dp)
- 内容：相机 / 相册 / 导入（从其他应用）

#### 交互说明

| 交互 | 行为 |
|------|------|
| 点击缩略图 | 进入编辑页，加载选中图片 |
| 长按缩略图 | 进入多选模式，可多选批量处理 |
| 下拉刷新 | 刷新相册列表 |
| 上滑加载 | 加载更多历史照片 |
| 点击相机 | 调用系统相机拍照 |
| 点击导入 | 打开系统文件选择器 |
